# Use Cases

Esta seção contém aplicações reais de IA.

## Exemplos

- chatbots corporativos
- sistemas de recomendação
- análise de documentos
- RAG aplicado
- automação com agentes

## Objetivo

Mostrar como arquiteturas e técnicas são aplicadas em problemas reais.

## Conteúdo esperado

- descrição do problema
- arquitetura da solução
- possíveis implementações
- variações

## Contribuição

Projetos devem ter foco em aplicação prática e contexto real.
